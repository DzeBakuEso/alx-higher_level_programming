## Python Data Structures Project

This repository contains solutions to various tasks involving Python data structures. The focus is on lists, tuples, and other data-related operations.

## Tasks

### 0. Print a list of integers
- **Prototype:** `def print_list_integer(my_list=[]):`
- Function that prints all integers in a list, one per line, using `str.format()`.

### 1. Secure access to an element in a list
- **Prototype:** `def element_at(my_list, idx):`
- Retrieves an element from a list at a given index. Returns `None` for invalid indices.

### 2. Replace element
- **Prototype:** `def replace_in_list(my_list, idx, element):`
- Replaces an element in a list at a specific position without modifying the original list.

### 3. Print a list of integers in reverse
- **Prototype:** `def print_reversed_list_integer(my_list=[]):`
- Prints all integers of a list in reverse order, one per line, using `str.format()`.

### 4. Replace in a copy
- **Prototype:** `def new_in_list(my_list, idx, element):`
- Replaces an element in a list at a specific position without modifying the original list.

### 5. Remove all characters `c` and `C`
- **Prototype:** `def no_c(my_string):`
- Removes all occurrences of the characters `c` and `C` from a string and returns the new string.

### 6. Print a matrix of integers
- **Prototype:** `def print_matrix_integer(matrix=[[]]):`
- Prints a matrix of integers, row by row.

### 7. Add 2 tuples
- **Prototype:** `def add_tuple(tuple_a=(), tuple_b=()):`
- Returns a tuple with the sum of the first two elements of the two input tuples.

### 8. More returns!
- **Prototype:** `def multiple_returns(sentence):`
- Returns a tuple with the length of a string and its first character.

### 9. Find the max integer
- **Prototype:** `def max_integer(my_list=[]):`
- Finds the biggest integer of a list. Returns `None` if the list is empty.

### 10. Find all multiples of 2
- **Prototype:** `def divisible_by_2(my_list=[]):`
- Returns a list of `True` or `False` based on whether the elements of the input list are divisible by 2.

### 11. Delete at a specific index
- **Prototype:** `def delete_at(my_list=[], idx=0):`
- Deletes the item at a specific position in a list. If the index is invalid, nothing changes.

### 12. Switch values of `a` and `b`
- **Source file:** `12-switch.py`
- Switches the values of two variables `a` and `b` in exactly 5 lines of code.

### 13. Linked list palindrome
- **Prototype:** `int is_palindrome(listint_t **head);`
- Checks if a singly linked list is a palindrome.
